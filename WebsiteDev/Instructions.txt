- To deploy site:
    1) If firebase-tools is not installed, go install it by following the instructions below titled "To install & login 
    to firebase-tools", then return step two
    2) Navigate to Help/WebsiteDev in the terminal
    3) Run "npm run build"
    4) Run "firebase deploy"

- To install & login to firebase-tools
    1) Open a terminal enter the command "npm install -g firebase-tools"
    2) Type in the command "firebase login"
    3) Login with the helpcocontact@gmail.com information
